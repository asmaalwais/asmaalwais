Projects

Selected academic and personal work

Smart Factory Automation

PLC, SCADA, MES, Automation

Designed and programmed a smart factory line controlled by PLC, monitored through SCADA, and coordinated via MES. The system manages multiple assembly stations, synchronizes production steps, and provides real time status feedback.

Roboter Auto

Embedded, Robotics, Sensors

Built and programmed an autonomous robot car capable of following paths and reacting to sensor input. Integrated motor control, sensor fusion, and basic decision logic on a microcontroller platform.

Regelungssystem PID

Control, PID, System modeling

Developed and tuned a PID controller to meet specific dynamic requirements. Modeled the system behavior, implemented the controller, and validated stability, overshoot, and response time through experiments.

Automatisiertes Smart Home

IoT, Sensors, Automation

Designed an automated smart home prototype with fire detection, security, automatic lighting, door pin access, and HVAC control. Combined sensors, actuators, and control logic into one integrated system.

Landwirtschaftlicher Roboter

Robotics, Computer Vision, Inverse kinematics

Developed an agricultural robot with a six axis robotic arm to remove dead leaves from strawberry plants. Used image processing to detect target leaves and inverse kinematics to plan arm motions.

Datenanalyse eines Schwingungssensors

Data analysis, Sensors, Python

Collected and analyzed vibration sensor data to study system behavior under different conditions. Applied signal processing and data visualization to identify patterns and anomalies.

AI Image Caption Generator

CNN, RNN, Deep learning

Implemented a neural network that generates text descriptions for images. Used a convolutional network for visual feature extraction and a recurrent network to produce natural language captions based on the image content.
